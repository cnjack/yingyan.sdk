## 百度鹰眼
百度鹰眼是一套集轨迹追踪、存储、运算、查询的完整轨迹开放服务，可帮助开发者管理多达100万人/车轨迹。   
使用百度鹰眼，您可以轻松开发出适用于车队监控、车联网、专车计费、外业人员监管、儿童防丢领域的轨迹管理产品。
